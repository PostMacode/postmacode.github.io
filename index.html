Projects

Live Piano

Creator · Link

Live piano program that can be played with a computer keyboard. I developed this program in C++ using the SDL library for graphics and audio. Programmed buffer size and sampling rate of the audio output to optimize for lowest possible latency. Measured the audio latency on my system to be a median of 4.7ms using a CRT oscilloscope, this measurement beats out Ableton Live's piano program by 12-14ms. In the future I plan to add different types of sounds so the user can play with more than just a generic synth piano and grand piano. I also plan on using the Gainput C++ library to allow the user to play with their USB devices, such as MIDI keyboards.

Global Terrorism Database

Creator · Link

Web app that uses a global database of 170,000 terrorism records from 1970 to 2015. You can sort by attack type, target type, weapon type, intent, related incidents, and more. Web interface made with R and Shiny. Database was obtained on Kaggle, cleaned with Jupyter and Python, then modeled in MySQL. Interface app connected to database through RJDBC. Database hosted online with Amazon Web Services and their Relational Database Service. I started this project to improve my understanding of databases. I used this as an opportunity to learn R and improve at Python and SQL. Previously the database was hosted on a Linux server but I switched to Amazon RDS because it's free and easier to modify in the future.
