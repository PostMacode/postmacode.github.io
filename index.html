Global Terrorism Database

Creator · Link

Web app that uses a global database of 170,000 terrorism records from 1970 to 2015. You can sort by attack type, target type, weapon type, intent, related incidents, and more. Web interface made with R and Shiny. Database was obtained on Kaggle, cleaned with Jupyter and Python, then modeled in MySQL. Interface app connected to database through RJDBC. Database hosted online with Amazon Web Services and their Relational Database Service. I started this project to improve my understanding of databases. I used this as an opportunity to learn R and improve at Python and SQL. Previously the database was hosted on a Linux server but I switched to Amazon RDS because it's free and easy to modify in the future.
